installation
Install coala without any bears

$ pip3 install coala

Install development version of coala

$ pip3 install coala-bears --pre

Installing inside a virtualenv

$ pip3 install virtualenv

$ virtualenv venv

$ source venv/bin/activate

On Unix based systems

$ source venv/bin/activate

On Windows

$ venv\scripts\activate

getting started

$ sudo pip3 install coala-bears

$ cat sum.py

if i==1:

    print ("Yes")

$ coala --files "**.py" --bears PEP8Bear --save

$ coala
